虚拟机安装 macOS 11 Big Sur Beta 体验版

· 2021-06-02 ·

VMware Workstation Pro ：https://www.vmware.com/products/workstation-pro/workstation-pro-evaluation.html 相关激活密钥请自行网上搜索试用。
macOS 11Big Sur Beta CDR镜像：https://pan.baidu.com/s/1Cy3j5qotEBLH1GzTQRF41Q 提取码: fpj8
必须核对SHA-2：11B8E1904F181E55D56532232D83795CDABB976F6EBBA980A1290778FF95136
为什么核对哈希值：百度网盘下载大文件有一定概率损坏文件，文件损坏后导致用户在安装部署时可能失败。

一、安装Unlocker（自己去找安装包吧，实在找不动了）
1.正常安装虚拟机并激活然后打开任务管理器将VMware的服务项全部停止，选中后点击右键即可看到停止。
2.下载并完整解压 Unlocker 至桌面，然后右键点击win-install.cmd并选择使用管理员权限运行这个批处理。
3.请注意本工具运行过程中需要连接VMware官网下载部分组件，因网络问题可能下载时间较长请耐心等待。
4.本工具成功下载辅助组件并解锁VMware后会自动启动相关服务，如果出现异常用户也可以直接重启系统。
5.完成后打开VMware虚拟机点击文件新建虚拟机选项，然后先不要挂载镜像选择稍后再提供操作系统镜像。
6.接下来在选择操作系统时就可以看到苹果系统的选项啦，配置完毕后再挂载苹果系统镜像即可执行安装喽。

二、安装配置
类型选择高级
选择镜像是把下方CD-ROM镜像改成所有文件
打开cdr文件
选择安装位置
操作系统选MAC OS X10.16
虚拟机名称和位置都可以自选
处理器核心以及内存方面越大越好，不然虚拟机会非常卡
分配内存
网络选择第二项
I/O控制器选第二项
磁盘选择SATA,创建新虚拟磁盘，并分配大小
选择磁盘文件位置
打开刚刚我们选择的虚拟机安装目录，比如蓝点网选择的是D:\VMware\Mac，然后找到 虚拟机名称.vmx 文件。
使用记事本或代码编辑器打开 VMX 文件，然后在结尾粘贴下面这三行代码，粘贴完成后直接保存配置文件即可。
smbios.reflectHost = "TRUE"
hw.model = "MacBookPro16,1"
board-id = "Mac-E1008331FDC96864"

二、配置磁盘
1.完成上述修改配置后即可开启虚拟机，接下来我们需要按照以下步骤对虚拟机进行配置，然后才能进入安装过程。
2.语言选简体中文
3.选择磁盘工具
4.在这里选择VMware虚拟硬盘并点击抹掉
5.将格式修改为APFS
6.完成抹掉后点击左上角磁盘工具退出
7.好了这里点击安装macOS进入安装进程
8.选择我们刚刚抹掉的硬盘即可，接下来安装会持续半个小时左右请耐心等待

三、收尾工作
安装完成后配置好一大堆选项
进入桌面后点击右上角Install macOS Beta选择推出
点击VMware虚拟机上的虚拟机---安装VMware Tools然后会弹出如下界面，双击安装
安装过程会触发多个安全限制，请按照提示转到设置里的安全设置里解锁，最后完成安装。
完成安装后必须重启系统，然后点击虚拟机上的进入全屏模式，macOS即可全屏显示
恭喜宝宝们完成了安装！
